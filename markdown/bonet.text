# BoNet

### From Bricowiki

Jump to: [navigation](#column-one), [search](#searchInput)
BoNet is a community-driven laboratory that researches and
practices co-existance in a more intense and sensitive manner.

[Bonete - IlhaBela](http://estudiolivre.org/el-gallery_view.php?arquivoId=5430 "http://estudiolivre.org/el-gallery_view.php?arquivoId=5430")

BoNet has always been a BricoLab interested in weather, sensor
networks, co-existance, plants, connectivity infrasctructure like
the BricoPhone, Internet Satelite Connection and Wifi-Mesh
technology as well as many new media art projects, including
"carbon kidnapping".
